Metroid Zero Mission - Unhundred%

Unhundred% is a hack where you start with every item in the game and remove them from your inventory by visiting the original item locations. You start in the room next to Mecha Ridley. The game ends when you touch morph ball with no other items. If you have any other items when you touch morph ball, it triggers a game over.

Softlocks are possible, and you can get into situations where items can no longer be reached. You'll need to plan ahead to avoid getting stuck.

The following changes were made to the game:
- Mother Brain and Ruins Test are already considered defeated. This lets you collect the items in Tourian without having to fight Mother Brain and do the escape.
- Some gerons that block your path have been removed, as well as the rippers in Norfair's entrance. This makes the game a bit easier and allows for more varied routing.
- Chozo statue hints are disabled.
- Various cutscenes are removed.
